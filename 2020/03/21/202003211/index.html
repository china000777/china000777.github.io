VScode常用快捷键，持续补充……

发表于 2020-03-21 分类于 技术

#VScode常用快捷键
####1、注释：
　　a) 单行注释：
1、 [先按ctrl+k，再按ctrl+c]
2、 ctrl+/

　　b) 取消单行注释：按下ctrl不放，再按k + u

　　c) 多行注释：
1、[alt+shift+A] 这种注释方式为


2、 选中要注释到代码段，按单行注释到快捷键

####2、显示/隐藏左侧目录栏:
ctrl + b

####3、删除行：
ctrl + shift + k

####4、控制台终端显示与隐藏：
ctrl + ~

####5、查找文件/安装vs code 插件地址：
ctrl + p

####6、行增加缩进:
ctrl + [

####7、行减少缩进:
ctrl + ]

####8、字体放大/缩小:
ctrl + ( + 或 - )

####9、拆分编辑器 :
ctrl + 1/2/3

####10、自动换行 :
alt + z(和txt文本到自动换行类似)

####11、全局查找文件：
ctrl + shift + f

####12、快速切换主题：
ctrl + k 再按 ctrl + t

####13、格式化选定代码 ：
ctrl + k 再按 ctrl +f

####14、多行同时添加内容（光标） ：
ctrl + alt + up/down

####15、打开最近打开的文件：
ctrl + r

####16、打开新的命令窗：
ctrl + shift + c
